## 📋 What Was Added

### 1. **Smart X1 - Bảo vệ Phần mềm Cơ bản**
- **Type**: Smart Card-based dongle
- **Storage**: 20KB total (4KB bulk + 64 pages × 256-byte)
- **Key Features**:
  - 3DES encryption (all operations inside dongle)
  - Hierarchical access control (Supervisor/User PIN)
  - 3 encryption methods (Direct API, Envelope, Combined)
  - Remote upgrade support
  - No driver required (Plug & Play)
- **Use Cases**: PC software protection, desktop applications, banking, government
- **Languages**: C, C++, VB, Delphi, Java, PowerBuilder, C#, VB.Net, ASP.Net

### 2. **Smart X3 - Bảo vệ Phần mềm Nâng cao (Programmable)**
- **Type**: ARM 32-bit programmable dongle
- **Storage**: 32KB data + 64KB executable (Professional version)
- **Performance**: 
  - 246x faster than traditional dongles
  - Execution time: < 0.1ms (vs ~20ms traditional)
  - Code size: ~2KB (vs ~20KB traditional)
- **Key Features**:
  - Direct code execution on ARM chip
  - Integrated encryption (RSA, 3DES, SHA1, MD5)
  - Periodic execution support
  - Shared memory between executables
  - Anti-tampering protection
- **Use Cases**: High-security software (AAA games, CAD, ERP), reverse engineering protection
- **Versions**: Standard vs Professional

### 3. **Smart UDisk - Secure Storage + Software Protection**
- **Type**: Flash Disk (16-32GB) + Dongle hybrid
- **Partitions**:
  - Flash Disk (normal access)
  - Safe Flash Disk (authenticated access)
  - Hidden Disk (encrypted, invisible)
  - CD Drive (software distribution)
- **Features**:
  - Software binding (copied software won't run elsewhere)
  - Two-factor authentication
  - Dynamic passwords
  - Virus protection
- **Use Cases**: Secure software distribution, sensitive data storage

### 4. **Smart Time Pro - Time Management & Licensing**
- **Type**: Smart Card-based time dongle
- **Features**:
  - Independent clock (USB-powered, tamper-proof)
  - Multiple time controls (usage time, expiration)
  - Flexible licensing (Trial, Leasing, Subscription)
- **Use Cases**: Trial software, subscription management, time-limited access
